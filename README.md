# newsletter_challengue
Frontend Mentor challengue to enhance skills creating forms with success message using React

### Core features:
1. User click on submit button to suscribe mail to newsletter form.
2. Confirm succesful suscription process on another view with the option to return home view once user click on "dismiss message" button.

### Teck stack: 
* React.js + Vite 
* npm packages

### Learnings:
1. Using Hook useNavigation to redirect another view when an event was executed.
2. Transfer content between 2 different .jsx files using useLocation hook.
3. Code events with handlers function.
4. Responsive design layout with mobile first principle.

### Usage

1. Clone repository
```bash
git clone git@github.com:BQC21/newsletter_challengue.git
``` 

2. Installl dependencies
```bash
npm install
``` 

3. Run web app
```bash
npm run dev
```

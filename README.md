# newsletter_challengue
Frontend Mentor challengue to enhance skills creating forms with success message using React

### Core features:
1. User click on submit button to suscribe mail to newsletter form.
2. Confirm succesful suscription process on another view with the option to return home view once user click on "dismiss message" button.

### Teck stack: 
* React.js + Vite 

### Learnings:
1. Using Hook useNavigation to redirect to another view when an event was executed.
2. Transfer content between 2 different .jsx files by useLocation hook.
3. Code events with handlers.
4. Responsive design layout with mobile first principle.
